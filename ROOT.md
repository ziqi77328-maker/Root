# Root 世界观总纲

## Root 是什么

Root 不是故事。

Root 是一片已经存在的森林。

它不会被建造，也不会被完成。世界始终处于生长之中。新的生命、新的地方、新的规则，都不是为了剧情被添加进来，而是在世界成长到合适的时候被发现。

Root 的工作不是创造森林，而是不断确认：森林一直都是这样。

## 根法则

### 1. 森林只有一个

Root 中永远只有一片森林。

森林不会复制，也不会因为不同生命的进入而产生新的森林。所有方向始终属于同一片森林。

### 2. 方向因“我”而存在

森林本身没有方向。

当“我”与 Root 建立连接，方向便因此诞生。不同的“我”拥有不同方向，但所有方向都属于同一片森林。

### 3. 每一个方向都会拥有一个使用者

拥有“我”，便成为森林使用者。

每一个使用者都会拥有属于自己的方向，也都会认为自己看见的是完整的森林。他们都没有错，因为每一个方向都是完整的一片森林。

### 4. 每一个方向都是完整的一片森林

方向不是另一个世界。

方向只是“我”看见森林的方式。不同方向中的树、路、重力、火把或脚下可能不同，但它们始终属于同一片森林。

### 5. 森林拥有变化，观察者命名时间

森林始终拥有变化。

观察者出现以后，观察者将森林持续发生的变化命名为时间。

因此，时间不是森林本身拥有的事物，而是观察者理解森林变化的一种语言。当最后一个观察者离开森林时，“时间”这个名字随之消失。森林并没有停止变化。

停钟塔开始以年轮记录这些持续发生的变化。

### 6. Root 只有三种最终归属

Root 中的一切最终只属于：

1. 森林
2. 方向
3. “我”

任何新存在如果无法归入其中之一，就不属于 Root。

## v2.0 补充发现

v2.0 不建立第二个 Root。它补充并校准的是森林如何回应、生命如何属于森林、连接如何成立、方向为何不保存痕迹，以及时间与年轮的关系。

已通过 Validator 的补充：

- 森林没有意识，不会思考、选择或回答问题。
- 森林的回应不是主动回答，而是自然规律持续发生。
- 森林中的生命属于森林；拥有“我”的生命成为观察者。
- 观察者不是新的生命种类，而是拥有“我”之后的生命状态。
- 连接不是创造，也不是改变；连接始于发现。
- 方向不会成长，也不会保存痕迹；真正成长与记住的始终是森林。
- 痕迹被森林淡忘时不是循环回来，而是以另一种形式参与新的诞生。
- 时间属于观察者对森林变化的命名，最终归入“我”；年轮属于森林。
- 每一个“我”都是唯一的，所有“我”属于同一种存在方式。
- 看见不等于发现。无“我”的生命可以看见鸟笼，但无法真正发现鸟笼。
- 小路是观察者留下的痕迹被另一位观察者重新发现之后形成的存在。

## 第一阶段底层规律

森林始终存在。

观察者始终发现。

发现建立连接。

连接留下痕迹。

痕迹最终回归森林。

## 禁止误读

- Root 不是平行宇宙。
- 方向不是世界。
- 鸟笼不是传送门，也不是另一个世界。
- 倒影湖不是跨世界连接。
- 停钟塔不让时间停止。
- 痕迹不是留言、证明或指引。
- Root 中没有神、预言、命运、系统或万能解释。
