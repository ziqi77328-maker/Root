# 版本记录

## v2.0 · 2026-07-07

状态：当前唯一版本

处理方式：

- v2.0 作为补充发现进入同一个 Root。
- v2.0 不建立第二个世界，也不推翻 v1.0 根法则。
- 与 v1.0 冲突的表述不进入正式文档，转入 [ideas.md](ideas.md) 待校准。

新增确认：

- 森林没有意识，不主动回答，只按自然规律持续发生。
- 森林记住痕迹，也会淡忘痕迹。
- 淡忘不是循环，旧痕迹会以另一种形式参与新的诞生。
- 生命属于森林；拥有“我”的生命成为观察者。
- 连接始于发现，不是创造或改变。
- 方向不会成长，也不会保存痕迹。
- 倒影湖作为自然湖，被发现了映照不同方向中“我”的规律。

拒绝或暂缓：

- 暂缓“时间属于观察者，森林没有时间”。
- 暂缓“所有观察者拥有同一个‘我’”。
- 拒绝“无‘我’的生命可以看见或进入鸟笼”。

## v1.0 · 2026-07-04

状态：根法则基准

建立内容：

- 确认 Root 不是小说、游戏或设定集，而是一片已经存在的森林。
- 确认森林只有一个。
- 确认方向因“我”而存在。
- 确认每一个方向都会拥有一个使用者。
- 确认每一个方向都是完整的一片森林。
- 确认所有方向共享森林时间。
- 确认 Root 中所有存在最终只属于森林、方向或“我”。
- 建立 Root Validator，作为所有新设定进入仓库前的检查入口。
- 建立正式文档、待确认文档与发现记录。

版本约束：

- Root 永远只有一个当前版本。
- 后续记录只能补充发现、校准表述或拒绝冲突设定。
- 不允许建立互相冲突的平行版本。
