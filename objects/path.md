# 小路

归属：森林或方向，待定

状态：未完善

## 当前确认

小路在旧文中作为痕迹例子出现过。

因此当前可以确认：

- 某个使用者走过后留下的小路，可以成为痕迹。
- 一旦成为痕迹，它属于森林。
- 后来的使用者可能发现这条小路，却不会遇见留下它的人。

## 待判断

具体小路也可能属于方向，因为不同方向里的小路可能不同。

需要区分：

- 方向中的路径感：可能属于方向。
- 留下并被后来发现的小路：属于森林，作为痕迹。

## 不允许

在通过 Validator 前，小路不能被写成必然指引、命运路线或系统路线。
